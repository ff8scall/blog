## 소개

AI 에이전트 스킬 관리는 복잡한 시스템과 여러 기능을 다루는 경우 특히 어렵습니다. 이 가이드에서는 `npx skills` CLI를 사용하여 AI 에이전트 스킬을 관리하는 방법을 살펴보겠습니다. 또한 이 가이드에서는 `npx skills`의 특이점과 오류를 설명합니다.

## `npx skills`란?

`npx skills`는 AI 에이전트 기능을 위한 npm-like 패키지 매니저로 작동하는 CLI 툴입니다. AI 에이전트 스킬을 설치, 업데이트 및 관리할 수 있으므로 복잡한 시스템을 배포하고 유지 관리하는 것을 더 쉽게 만듭니다.

## `npx skills` 설치

`npx skills`를 사용하기 위해 시스템에 설치해야 합니다. 다음 명령어를 터미널에 입력하여 설치할 수 있습니다.

```bash
npm install -g @skills/cli
```

설치가 완료되면 다음 명령어를 실행하여 `npx skills`가 올바르게 작동하는지 확인할 수 있습니다.

```bash
npx skills --version
```

## `npx skills` 사용

`npx skills`가 설치되면 이제 사용해 보겠습니다. 다음은 기본 명령어를 사용하여 시작하는 방법입니다.

* `npx skills list`: 사용 가능한 스킬 목록을 표시합니다.
* `npx skills install <skill-name>`: 특정 스킬을 설치합니다.
* `npx skills update`: 설치된 모든 스킬을 업데이트합니다.
* `npx skills uninstall <skill-name>`: 특정 스킬을 삭제합니다.

## 특이점과 오류

`npx skills`는 강력한 툴이지만 특이점과 오류가 있습니다. 다음은 주의해야 할 사항입니다.

* `npx skills`는 Node.js 환경이 필요합니다. 시스템에 Node.js가 설치되어 있지 않으면 `npx skills`를 사용할 수 없습니다.
* 일부 스킬은 올바르게 작동하기 위해 추가 의존성이나 구성이 필요할 수 있습니다. 각 스킬의 문서를 읽어보는 것을 잊지 마십시오.
* `npx skills`는 package.json 파일을 사용하여 스킬을 관리합니다. 스킬이 변경될 때마다 package.json 파일을 업데이트하십시오.
